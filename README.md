# Este repositório guarda todos os meus projetos web

## Como acessar os projetos no seu navegador:
Acesse o projeto que você quer ver e copie o caminho do projeto <br>
Após isso cole o caminho do projeto em uma nova aba do seu navegador. 

## Exemplo de acesso de projeto:
Neste exemplo estou utilando o projeto "projeto-portfolio-bs"
 - Na branch 'Master' do repositório clique sobre a pasta do projeto
 - copie o caminho do projeto: Ygor-Evaldt-dev.github.io/projeto-portfolio-bs
 - Cole em uma nova aba do seu navegador.
